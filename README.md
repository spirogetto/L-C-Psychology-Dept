# L-C-Psychology-Dept
💘 Love &amp; Chaos is a stealth puzzle game set inside a house full of secrets and bad decisions. Moin loves Sakia. Sakia is married to Jayanta. Jayanta is cheating with Adity. Moin also likes Adity. Everyone wants something — nobody is safe. Pick a character, navigate 6 rooms, complete your goals, and don't get caught.
